
PROGRAMOS IŠKVIETIMAS
=====================
Programa PRIVALO priimti du įvesties šaltinius: FASTA_FAILAS_1 ir FASTA_FAILAS_2.

Jei įvesties failo vardas yra vienas brūkšnelis ('-'), programa PRIVALO skaityti
atitinkamą seką iš STDIN.

Jei įvesties failų vardai nenurodyti, programa PRIVALO skaityti abi sekas iš STDIN.

Pavyzdžiai:
-----------
1) Dviejų failų palyginimas:
   nw-align seqA.fasta seqB.fasta

2) Antros sekos skaitymas iš STDIN:
   nw-align seqA.fasta - < seqB.fasta

3) Abiejų sekų skaitymas iš STDIN:
   nw-align - - < input.fasta


STDIN / STDOUT / STDERR ELGSENA
==============================
Jei įvesties failų vardai komandinėje eilutėje nenurodyti, programa PRIVALO
skaityti įvestį iš STDIN.

Programa PRIVALO rašyti visus rezultatus į STDOUT.

Visi klaidų pranešimai PRIVALO būti rašomi į STDERR.

Programa NETURI daryti prielaidos, kad STDIN yra pervyniojamas (rewindable).


PROGRAMOS FUNKCIJA
==================
Programa PRIVALO atlikti dviejų biologinių sekų globalų porinį
išlygiavimą, naudodama Needleman–Wunsch algoritmą.

Programa PRIVALO apskaičiuoti optimalų globalų išlygiavimą tarp dviejų
sekų, maksimalizuodama išlygiavimo balą pagal nurodytą balų (scoring)
schemą.

Programa PRIVALO sugeneruoti:
- galutinį išlygiavimo balą (score),
- dvi išlygiuotas sekas su tarpais ('-'),
- išlygiavimo ilgį.

Programa NETURI atlikti lokalaus išlygiavimo (Smith–Waterman) ar kelių
sekų išlygiavimo.


ĮVESTIES FORMATAS
=================
Programa PRIVALO priimti įvestį FASTA formatu.

Kiekvienas FASTA įrašas PRIVALO prasidėti antraštės eilute, prasidedančia
simboliu '>'.

Sekos simboliai PRIVALO būti pateikti sekančiose eilutėse ir GALI būti
išskaidyti per kelias eilutes. Tarpai ir tuščios eilutės TURĖTŲ būti
ignoruojami.

Programa PRIVALO naudoti tik pirmą FASTA įrašą kiekviename įvesties
šaltinyje. Papildomi FASTA įrašai TURĖTŲ būti ignoruojami, apie tai
pranešant kaip WARNING arba NOTE.

Leistini simboliai (minimalus palaikymas):
- DNR/RNR: A, C, G, T, U
- Baltymai: 20 standartinių aminorūgščių raidžių

Kiti simboliai PRIVALO būti traktuojami kaip klaida.


ALGORITMO APRAŠYMAS
===================
Programa PRIVALO naudoti Needleman–Wunsch dinaminio programavimo
algoritmą globaliam sekų išlygiavimui.

Tegu A = a1..am ir B = b1..bn – dvi įvesties sekos.

Programa PRIVALO sukurti (m+1) × (n+1) dydžio balų matricą F, kur:

F(0,0) = 0
F(i,0) = i × gap
F(0,j) = j × gap

Vidinės matricos reikšmės PRIVALO būti skaičiuojamos pagal formulę:

F(i,j) = max(
  F(i−1,j−1) + s(ai,bj),
  F(i−1,j)   + gap,
  F(i,j−1)   + gap
)

Kur:
- s(ai,bj) – simbolių sutapimo / nesutapimo balas,
- gap – tarpo (gap) bauda.

Programa PRIVALO atlikti atsekimą (traceback) nuo F(m,n) iki F(0,0),
sugeneruojant dvi išlygiuotas sekas A' ir B' su tarpais '-' taip, kad
A' ir B' ilgis sutaptų.


IŠVESTIES FORMATAS
==================
Programa PRIVALO rašyti rezultatus į STDOUT paprasto teksto formatu.

Išvestis PRIVALO turėti:
- galutinį išlygiavimo balą (Score),
- išlygiavimo ilgį (Length),
- dvi išlygiuotas sekas.

Rekomenduojamas minimalus formatas:
Score: <int>
Length: <int>
A: <aligned_A>
B: <aligned_B>

Pavyzdys:
---------
Score: 10
Length: 9
A: ACGTACGT-
B: ACGT-CGTG


NEPRIVALOMI ARGUMENTAI (OPTIONS)
================================
Programa PRIVALO palaikyti šias opcijas:

--help
  Išveda trumpą naudojimo informaciją į STDOUT ir baigia darbą.

--version
  Išveda programos versiją į STDOUT ir baigia darbą.

Programa GALI palaikyti papildomas opcijas balų schemai keisti:
--match INT       (default: 1)
--mismatch INT    (default: -1)
--gap INT         (default: -1)

Jei opcijos nerealizuojamos, tai PRIVALO būti aiškiai nurodyta README ir
šiame dokumente.
